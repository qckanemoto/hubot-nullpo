## Installation

Install via npm.

```bash
$ cd /path/to/hubot
$ npm install --save hubot-nullpo
```

And add to `external-scripts.json`.

```bash
$ cat external-scripts.json
["hubot-nullpo"]
```

## Configuration

```bash
# optional
$ export HUBOT_NULLPO_RICH_STYLE="true"

# if "true" then response will be like this:
#
# 　 Λ＿Λ　　　　＼＼
```
